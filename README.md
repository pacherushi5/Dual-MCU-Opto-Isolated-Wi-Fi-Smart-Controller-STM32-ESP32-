
# Dual-MCU Opto-Isolated Wi-Fi Smart Controller

This repository contains the complete hardware design of a
Dual-MCU Smart Controller using STM32 and ESP32.

The design emphasizes electrical safety, modularity,
and industrial-ready isolation using optocouplers.

## Design Highlights
- STM32 for deterministic real-time control
- ESP32 for Wi-Fi and IoT connectivity
- Opto-isolated MCU-to-MCU communication
- Isolated power domains
- PCB designed for noise immunity

## Applications
- Industrial automation

## Safety
- Logic-level isolation using optocouplers
- Separate ground domains
- Designed with EMI considerations
